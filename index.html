How it works

So funktioniert Ultimate Tic Tac Toe

Das Spiel besteht aus neun kleinen Tic-Tac-Toe-Feldern, die zusammen ein großes Spielfeld bilden.

01

Spiele in einem Mini-Board

Jeder Zug wird zuerst in einem der kleinen 3×3-Felder gesetzt.

02

Bestimme das nächste Board

Die Position deines Zuges entscheidet, in welches Mini-Board dein Gegner als Nächstes spielen muss.

03

Gewinne kleine Boards

Wenn du in einem Mini-Board drei in einer Reihe hast, gehört dieses Board dir.

04

Gewinne das große Spiel

Gewinne drei Mini-Boards in einer Reihe auf dem großen Raster, um das Match zu gewinnen.
